# E-Commerce AI

Full-stack e-commerce application with React frontend and Node.js/Express backend.

## Features

- User authentication (register/login)
- Product catalog
- Shopping cart
- Order management
- AI recommendations

## Tech Stack

- **Frontend:** React, Vite
- **Backend:** Node.js, Express, MongoDB
- **Authentication:** JWT

## Getting Started

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Environment Variables

Backend (.env):
- MONGO_URI
- JWT_SECRET
- PORT
